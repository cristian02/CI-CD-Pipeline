# CI-CD-Pipeline
Deploy API in Azure with github actions 

<p>Commands executed</p>
<ol>
  <li>dotnet --version </li> Check out your dotnet version 
  <li>dotnet new webapi -o Pipelines </li> Create and include web api in solution
  <li>dotnet build</li>
  <li>dotnet run</li>
  
</ol>
